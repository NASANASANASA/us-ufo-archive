The site is published through GitHub Pages with a custom domain. It provides an interactive archive interface and static SEO-friendly pages for individual records and index pages.

---

## What This Project Is

This repository contains a public archive mirror for U.S. government UFO / UAP release materials. The site organizes official record titles, metadata, descriptions, source links, documents, images, videos, and related archive pages into a searchable multilingual interface.

中文说明：

- 本站并非美国政府官方网站。
- 档案文字、文件、图片、视频与来源链接均基于美国政府公开发布资料整理。
- 中文及其他语言内容仅作翻译与信息整理。
- 如翻译、排版或元数据与官方来源不一致，以官方英文来源为准。

---

## Main Features

- Interactive archive database with search, filters, sorting, pagination, and record detail views.
- Static record pages for search engines, sharing, and direct linking.
- Individual URL for each archive record.
- Index pages by agency, year, location, and file type.
- Multilingual interface and translated archive metadata.
- Original-source links preserved for source verification.
- Sitemap and robots.txt generated for search indexing.
- Google Search Console, Google Analytics 4, and Google AdSense integration.

```text

/en/agencies/...      Agency indexes
/en/years/...         Year indexes
/en/locations/...     Location indexes
/en/types/...         File-type indexes
```

Equivalent paths exist for the supported core languages.

## Data Source

The archive is based on public U.S. government release material, especially the official UFO / UAP release portal:

https://www.war.gov/UFO/

Each record keeps the original source link when available. This project does not claim government affiliation, endorsement, or analytical authority over the records.

## Local Preview

From the repository root:

```bash
python3 -m http.server 9001
```

Then open:

```text
http://localhost:9001/
```

---

## Build / Maintenance

The static archive pages are generated by:

```bash
node scripts/build-static.js
```

Before publishing changes, run basic checks such as JavaScript syntax validation and link checks when applicable.
