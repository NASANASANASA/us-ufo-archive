This record is a PDF UAP-related asset released by FBI in Release 01. The incident date is listed as N/A, and the incident location is listed as N/A. This page preserves the official source link, original description, and structured metadata.

Official Description

The FBI's 62-HQ-83894 case file includes investigative records, eyewitness testimonies, and public reports concerning Unidentified Flying Objects and flying discs documented between June 1947 and July 1968. The records include high-profile incident accounts, photographic evidence from sites like Oak Ridge, TN, and technical proposals regarding potential propulsion systems. Additional topics include convention programs, researcher accounts, and extensive media coverage from the period. This file is partially posted on FBI vault with more redactions and some pages missing. Included here is the complete case file with several newly declassified pages and only minor redactions.
