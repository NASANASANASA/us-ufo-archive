CIA-UAP-013, Report of Unusual Flying Object Sightings and Attendant Scientific Activity

Registro del archivo público UAP del Gobierno de Estados Unidos: CIA-UAP-013, Report of Unusual Flying Object Sightings and Attendant Scientific Activity. Agencia: CIA. Fecha del incidente: 1956. Lugar del incidente: Budapest, Hungary.

Resumen estructurado

Este registro es un material relacionado con UAP en formato PDF, publicado por CIA en Publicación 03. La fecha del incidente figura como 1956 y el lugar como Budapest, Hungary. La página conserva el enlace a la fuente oficial, la descripción original y los metadatos principales.

Descripción oficial en inglés

This 1956 CIA Information Report describes flying object sightings reported by a Budapest-based sub-source, including a sketch showing the objects' suspected formation and flight path between Budapest and Moscow. A more redacted version of this report has been available on CIA's public website.
