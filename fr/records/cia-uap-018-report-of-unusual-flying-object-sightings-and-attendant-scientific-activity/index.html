CIA-UAP-018, Report of Unusual Flying Object Sightings and Attendant Scientific Activity

Enregistrement de l'archive publique UAP du gouvernement des États-Unis: CIA-UAP-018, Report of Unusual Flying Object Sightings and Attendant Scientific Activity. Agence: CIA. Date de l'incident: 1955. Lieu de l'incident: Hongrie.

Résumé structuré

CIA在Publication 03公开的 PDF 类型 UAP 相关档案。事件日期：1955；事件地点：Hongrie。本页保留官方来源链接、翻译说明与主要元数据。

Description officielle

Cette collection contient des rapports de lecture technique sur des drones/UAV et des PANI obtenus par le CENTCOM lors de diverses missions de soutien.
